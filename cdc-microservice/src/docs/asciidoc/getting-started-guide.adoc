[introduction]
== Introduction

CDC Microservice (Fraud) is a demonstrator RESTful web service that shows how to use Spring Cloud Contracts and Spring Rest Docs.


[getting-started]
== Getting started

[getting-started-running-the-service]
=== Running the service
CDC-Microservice (Fraud) is written using http://projects.spring.io/spring-boot[Spring Boot] which
makes it easy to get it up and running so that you can start exploring the REST API.

The first step is to clone the Git repository:

[source,bash]
----
$ git clone https://github.com/benwilcock/spring-cloud-contracts
----

Once the clone is complete, you're ready to get the service up and running:

[source,bash]
----
$ cd cdc-microservice
$ gradle build publishToMavenLocal bootRun
----
