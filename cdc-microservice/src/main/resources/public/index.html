You can write the docs using asciidoc (adoc) format. Spring REST Docs writes 'snippets' which you are free to `include` wherever you like in your adoc files.
Note: If you run the server from your IDE you may not get any docs being served because the regular build steps can get bypassed by the IDE. Just run `gradle clean build bootRun`.
